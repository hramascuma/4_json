# Prettify JSON

Represents json raw data in a readable form.

# Quickstart

Example of script launch on Linux, Python 3.5:
```bash
$ python pprint_json.py <path to file>
```
# Output example

```bash
[
  {
        "test": true,
        "netest": false,
        "numder": 0,
        "string": "Hello"
    }, {
        "test": true,
        "netest": false,
        "numder": 1,
        "string": "Hello"
    }
]
```
